# Grafo com Matriz de Adjacência em Python

Este projeto demonstra como representar um grafo usando uma matriz de adjacência
e visualizá-lo com a biblioteca NetworkX.

## Tecnologias usadas
- Python
- NetworkX
- NumPy
- Matplotlib

## Funcionalidades
- Criação manual de matriz de adjacência
- Conversão para grafo
- Visualização gráfica
- Consulta de adjacência entre nós

## Como executar
```bash
pip install networkx numpy matplotlib
python grafo.py
